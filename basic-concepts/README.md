# 基础概念

本章介绍 XR 技术的核心概念和基础知识，为后续学习打下坚实的理论基础。

## 📚 章节内容

### [XR 技术概述](xr-overview.md)
了解 XR 技术的发展历程、分类和应用场景。

### [Unity 基础](unity-basics.md)
掌握 Unity 引擎的基本操作和核心概念。

### [3D 数学基础](math-fundamentals.md)
学习 3D 开发必备的数学知识。

## 🎯 学习目标

完成本章学习后，您应该能够：
- 准确区分 VR、AR、MR 等不同类型的 XR 技术
- 熟练使用 Unity 编辑器的基本功能
- 理解 3D 空间中的坐标系统和变换
- 掌握向量、矩阵等数学工具的应用

## 🔍 学习建议

建议按照以下顺序学习本章内容：
1. 先阅读 XR 技术概述，建立整体认知
2. 学习 Unity 基础操作，熟悉开发环境
3. 最后深入学习数学基础，为实践做准备

每学完一个小节后，建议花时间复习和整理笔记。
