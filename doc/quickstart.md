## üöÄ Your First Program

Let's create a simple XR scene:

### Step 1: Create New Project
1. Open Unity Hub
2. Click "New Project"
3. Select "3D" template
4. Name your project and create

### Step 2: Add Basic Objects
1. Right-click in Hierarchy window
2. Select "3D Object" ‚Üí "Cube"
3. Adjust position and size in Inspector window

### Step 3: Add Camera
1. Create a new empty GameObject
2. Add "XR Rig" component
3. Configure camera settings

### Step 4: Run Test
1. Click Play button
2. Observe scene effects
3. Try different perspectives
