# Unity XR 开发指南

[English](README.md) | 中文

这是一个专为学生和初学者设计的 Unity XR 开发学习指南，涵盖从基础概念到项目实战的完整学习路径。

## 🎯 特色功能

- ✅ **结构化学习**: 循序渐进的知识体系
- ✅ **实践导向**: 丰富的代码示例和项目案例
- ✅ **多平台支持**: 兼容 VR/AR/MR 各种设备
- ✅ **在线阅读**: 基于 Docsify 的响应式文档系统
- ✅ **PDF 导出**: 支持离线阅读和打印
- ✅ **搜索功能**: 快速定位所需内容

## 🚀 快速使用

### 在线阅读
直接访问 GitHub Pages: https://your-username.github.io/unity-xr-guide

### 本地运行
```bash
# 克隆仓库
git clone https://github.com/your-username/unity-xr-guide.git
cd unity-xr-guide

# 安装依赖
npm install

# 启动本地服务器
npm run dev
```

然后在浏览器中打开 `http://localhost:3000`

### PDF 导出
```bash
# 生成 PDF 文档
npm run pdf
```
