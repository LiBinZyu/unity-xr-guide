# Unity XR 开发指南

欢迎来到 Unity XR 开发学习指南！本指南专为初学者和学生设计，帮助您从零开始掌握 Unity XR 开发技能。

## 🎯 学习目标

通过本指南的学习，您将能够：
- 理解 XR 技术的基本概念和发展趋势
- 掌握 Unity 引擎在 XR 开发中的应用
- 独立完成 VR/AR/MR 项目的开发
- 优化 XR 应用的性能表现
- 成功发布自己的 XR 作品

## 🚀 快速开始

### 在线阅读
访问 GitHub Pages: https://LiBinZyu.github.io/unity-xr-guide

### 本地运行
```bash
# 方法1: 使用启动脚本（推荐）
./start.sh        # macOS/Linux
start.bat         # Windows

# 方法2: 直接使用命令（无需全局安装）
npx docsify-cli@latest serve . --port 3000

# 方法3: 使用 npm scripts
npm run dev
```

### PDF 导出
使用浏览器打印功能导出为 PDF：
1. 打开在线文档
2. 按 Ctrl+P (Windows) 或 Cmd+P (Mac)
3. 选择"另存为 PDF"
